# Spotify Data Analysis using Python

## Project Overview

This project analyzes Spotify songs data using Python in Google Colab. It includes data cleaning, exploratory data analysis (EDA), and visualizations to identify trends in song popularity, artists, genres, and audio features.

## Tools Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Features

* Data Cleaning
* Handling Missing Values
* Removing Duplicates
* Summary Statistics
* Top Artists Analysis
* Genre Analysis
* Popularity Analysis
* Correlation Heatmap
* Data Visualization

## Files

* Spotify_Data_Analysis.ipynb – Google Colab notebook
* spotify.csv – Dataset (if included)
* README.md – Project documentation

## Conclusion

This project demonstrates the complete data analysis workflow using Python, from importing and cleaning data to creating visualizations and generating insights.
